Designer Bay Project

Overview

This project was created as part of a team whilst at Northumbria University. The team consisted of five members including myself with the overall aim being to design and develop a website that could be used by freelancers to advertise thier skills to get work. This information could be seen by clients who would be able to recruit a freelancer.

      The team members and thier roles in the project included:

  • Will Stewart - Communications
  • Ananta Handley - Client & Freelancer Profiles
  • Callum Whitehouse - Forum & Events
  • Dane Ridley - Logo
  • James Ashcroft - Admin Functionality

The Approach

This project was produced rapidly in only a couple of months, this was due to strict time constraints for completion. The team used an agile methodology throughout the design and development stages using a trello board to record the work of each team member. The designs produced included wireframes, usecase diagrams and site flow diagrams. The development was completed using HTML, CSS and PHP.
